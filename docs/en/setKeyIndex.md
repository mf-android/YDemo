```objective-c
public int setKeyIndex(int mainKeyIndex)
```

- #### Parameter
| Name | Type | Description |
| :-------- | :--------| :------ |
| mainKeyIndex | int | values are as follows 0 ~7 |

- #### Return
| Value | Description |
| :--------| :------ |
| 0 | set key index success |
| other | set key index fail    |
