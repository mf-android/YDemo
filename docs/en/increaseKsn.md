# increaseKsn

### Interface functions
> Get the last ksn or increment ksn

### Prototype

```objective-c
public String increaseKsn(boolean isIncrease)
```

- #### Parameter
| Name | Type | Description |
| :-------- | :--------| :------ |
| isIncrease | boolean | If set true increment ksn else get last ksn |

- #### Return
| Value | Description |
| :--------| :------ |
| String | Get ksn success |
| null | Get ksn fail |
